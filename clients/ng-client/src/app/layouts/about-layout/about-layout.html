About PrvtMssgs


PrvtMssgs is a lightweight, privacy-focused chat application designed for secure and anonymous communication.


To start chatting, users simply enter:


  • a room name (the group they want to join),
  • their username, and
  • a password used to encrypt and decrypt messages

This password never leaves the client. It is used locally to encrypt messages before they are send.


End-to-End Encryption


All messages in PrvtMessgs are encrypted on the client side before being send to the server.

The server never sees the actual message content - it only receives encrypted payloads and forwards them to the other users in the same room.


Because of this design, the server effectively acts as a "zero-knowledge relay" (sometimes referred to as a zero-room server): it simply routes encrypted messages without knowing their contents.


No Database, No Tracking


PrvtMssgs does not store messages on the server.


Messages exist only


  • in the memory of connected clients and
  • in the browser's session storage.

This means:


  • messages disappear when the browser tab is closed,
  • no chat history is stored on the server,
  • no personal data or accounts are required.

Anonymous by Design


The server does not know:


  • who the users are,
  • what the messages contain,
  • or any persistent identity.

it only forward encrypted messages between clients connected to the same room.


Open Source


PrvtMssgs is fully open source, so anyone can inspect the code, verify how encryption works, or run their own instance.


You can view the source code on GitHub.


Educational Project & Responsible Use


PrvtMssgs was created primarily as a learning project to explore topics such as real-time communication, encryption in the browser, and privacy-focused application design.


This project is not intended to support or encourage any illegal or harmful activities. Users are expected to use the application responsibly and in accordance with applicable laws.


The goal of this project is to demonstrate how modern web technologies can be used to build secure, privacy-respecting communication tools.
